## useDebounce

```js
import { useDebounce } from "@ds-tools/design-system/core/hooks";
```

This hook is to propose a JS managed focus state.

```js
function useDebounce(callback, timeout = USE_DEBOUNCE_DEFAULT_TIMEOUT, options) => func;
```

-   callback `func`: function to debounce
-   time `number`: debounce value
-   options `object`: options
-   returns `func`: debounced function
