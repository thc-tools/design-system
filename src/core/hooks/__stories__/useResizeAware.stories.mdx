## useResizeAware

```js
import { useResizeAware } from "@ds-tools/design-system/core/hooks";
```

This hook is to help detect current size of an element

```js
function useResizeAware() => [resizeListener, sizes];
```

-   return `array`:
    -   resizeListener `node`: node to put in the element we want size information from
    -   sizes `{ heigt: number, width: number }`: Current sizes of element
