# Cursor

## TODO

Cursor in ds have four defined quarks:

-   `--ds-cursor--pointer`: is used when an element have a click interaction
-   `--ds-cursor--notAllowed`: is used when an element is disabled
-   `--ds-cursor--text`: if the element is editable
-   `--ds-cursor--default`: to reset to the default cursor if nested in another
