# Borders

## Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

### Borders

Border in ds have three defined quarks:

-   `--ds-borderSolid--S`
-   `--ds-borderSolid--L`
-   `--ds-borderDashed--S`

They can be used with radius and color quarks.

By default the border will use the `--ds-textColor--secondary` for its color.

### Radius

Radius in ds have four defined quarks:

-   `--ds-radius--XS`: 3px
-   `--ds-radius--S`: 5px
-   `--ds-radius--M`: 10px
-   `--ds-radius--L`: 100px
