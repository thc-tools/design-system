# Animation

Three durations exists to animate your transitions :

-   `--ds-animationDuration--S`: 300ms
-   `--ds-animationDuration--M`: 500ms
-   `--ds-animationDuration--L`: 700ms

To animate a property you can use the CSS `transition` property.

Reference: https://developer.mozilla.org/fr/docs/Web/CSS/transition

⚠️ Do not forget to specify which property to apply a transition to.
Otherwise, every property (even on nested elements) will have a transition,
it is certainly not what you want.

```CSS
.ds-c-my_superb_class {
    background-color: white;

    transition: background-color var(--ds-animationDuration--S);
}

.ds-c-my_superb_class:hover {
    background-color: pink;
}
```

You can also read: https://material.io/design/motion/speed.html#easing
